// how it works

From raw web to risk score

Four layers turn live internet data into actionable underwriting signals.

01 // CRAWL

Live Web Crawler

Async crawler hits 15 pages simultaneously across news, finance, gov, and research domains. Follows links up to 3 hops from seeds.

02 // EXTRACT

NLP Entity Extraction

spaCy named-entity recognition pulls every Person, Organization, Location, Product, and Financial Figure from each page.

03 // SCORE

3XB Weighted Tagging

Sigmoid normalization blends base weight + frequency signals. Temporal decay keeps scores fresh — stale data fades automatically.

04 // GRAPH

Relationship Propagation

Co-occurrence edges build an entity graph. Risk propagates 2 hops — a founder's connected companies affect their composite score.

// core product

Loan scoring, web-native.

One API call returns a composite risk score built from entity weight, neighbor graph, and temporal signals.

Pass any borrower entity name. 3XB returns their web prominence weight, 2-hop neighbor risk average, and a composite APPROVE / REVIEW / DECLINE verdict.
